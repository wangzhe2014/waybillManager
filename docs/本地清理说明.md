# 本地清理说明

## diagnostics 临时接口

`src/app/api/diagnostics/*` 是联调排障阶段创建的临时接口，不属于最终交付功能。当前普通删除命令返回 `Access is denied`，因此需要在 Windows 管理员 PowerShell 中手工清理。

请在管理员 PowerShell 中执行：

```powershell
$paths = @(
  'D:\trae\waybillManager\src\app\api\diagnostics\rules-enabled\route.ts',
  'D:\trae\waybillManager\src\app\api\diagnostics\v2\route.ts',
  'D:\trae\waybillManager\src\app\api\diagnostics\v2-env\route.ts',
  'D:\trae\waybillManager\src\app\api\diagnostics\v2-fetch\route.ts',
  'D:\trae\waybillManager\src\app\api\diagnostics\v2-http\route.ts'
)

foreach ($path in $paths) {
  if (Test-Path -LiteralPath $path) {
    takeown.exe /F $path /A
    icacls.exe $path /grant "$env:USERNAME:F"
    Remove-Item -LiteralPath $path -Force
  }
}

$dirs = @(
  'D:\trae\waybillManager\src\app\api\diagnostics\rules-enabled',
  'D:\trae\waybillManager\src\app\api\diagnostics\v2',
  'D:\trae\waybillManager\src\app\api\diagnostics\v2-env',
  'D:\trae\waybillManager\src\app\api\diagnostics\v2-fetch',
  'D:\trae\waybillManager\src\app\api\diagnostics\v2-http',
  'D:\trae\waybillManager\src\app\api\diagnostics'
)

foreach ($dir in $dirs) {
  if (Test-Path -LiteralPath $dir) {
    Remove-Item -LiteralPath $dir -Force
  }
}
```

确认命令：

```powershell
Test-Path -LiteralPath 'D:\trae\waybillManager\src\app\api\diagnostics'
```

期望输出：

```text
False
```

说明：`tsconfig.json` 目前排除了 `src/app/api/diagnostics/**`，只是为了避免临时诊断接口影响业务类型检查。最终提交前仍建议物理删除该目录。
